📖 Cara Edit di Visual Studio Code

  1. Buka VS Code → File → Open File → pilih bani-maruf.html
  2. Tekan Ctrl + F untuk mencari nama yang ingin diedit
  3. Edit langsung: nama, tanggal lahir, alamat, dll. pada bagian var familyData = {...}
  4. Untuk simbol almarhum: cari alm: true atau field death
  5. Simpan dengan Ctrl + S lalu refresh browser
  6. Gunakan Ctrl + Shift + F untuk format ulang kode otomatis
